# DevOps Backend Project

This is my first backend project.

## What this project does
- Runs a backend server
- Shows message on browser
- Provides API at /users

## How to run
1. Install Python
2. Install Flask
3. Run: python app.py
4. Open: http://127.0.0.1:5000

## API
- / → shows app running
- /users → returns user data

## Tech used
- Python
- Flask
